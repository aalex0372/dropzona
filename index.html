Step 1: Connect Twitch

Step 2: Steam Trade Bot

The bot account sends trade offers to winners. Create a dedicated Steam account for the bot.

Step 3: CS2 Game Agent

The agent captures game events (ACE, Triple Kill, Clutch) and sends them to DROPZONE. Without it, drops won't work.

CS2 Game State Integration
Add config to your CS2 cfg folder
// gamestate_integration_dropzone.cfg
"DROPZONE" {
  "uri" "https://api.dropzone.gg/events"
  "timeout" "5.0"
  "data" { "player_id" "1" "player_state" "1" "round" "1" "map" "1" }
}
